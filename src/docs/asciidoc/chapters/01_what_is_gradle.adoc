== What is Gradle?

=== Gradle's purpose

Gradle Build Tool is a build and automation tool

* JVM based
* Implemented in Java
* 100% Free Open Source - Apache Standard License 2.0

=== Agnostic Build System

* JVM ecosystem
** Java, Kotlin, Groovy, Scala, Clojure ...
* Native ecosystem
** C, C++, Swift, ...
* Android
* Misc
** Go, Python, JavaScript, Asciidoctor, ...
